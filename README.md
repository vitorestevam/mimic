# Mimic

Mimetize sensible data

## How to use

-f Refers to the file you want to mimetize

-r Refers to the rules you want to apply

```
./mimic.exe -f ./file -r uuid date
```

## Available rules

| Rule    | Example |
| -------- | ------- |
| uuid  | b40aa1d8-35b3-8cca-bbce-98264a8220d0 |
| datetime | 2045-01-01 19:44:30 |
